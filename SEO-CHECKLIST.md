# SEO Optimizasyon Checklist

## ✅ Tamamlanan SEO İyileştirmeleri

### 1. Meta Tags ve HTML Head
- ✅ Title tag optimize edildi (60-70 karakter)
- ✅ Meta description eklendi (150-160 karakter)
- ✅ Meta keywords eklendi
- ✅ Robots meta tag
- ✅ Language ve revisit-after meta tags
- ✅ Canonical URL

### 2. Open Graph (Social Media)
- ✅ Facebook Open Graph tags
- ✅ Twitter Card tags
- ✅ OG image, title, description

### 3. Schema.org Structured Data
- ✅ JSON-LD yapılandırılmış veri
- ✅ TransportationCompany schema
- ✅ Service offerings schema
- ✅ Address, phone, email bilgileri
- ✅ Opening hours
- ✅ Aggregate rating

### 4. Site Dosyaları
- ✅ robots.txt oluşturuldu
- ✅ sitemap.xml oluşturuldu

### 5. Semantic HTML
- ✅ Main content area `<main>` tag ile sarmalandı
- ✅ Header `role="banner"` eklendi
- ✅ Nav `aria-label` eklendi
- ✅ Footer `role="contentinfo"` eklendi
- ✅ Sections için `aria-label` eklemeleri

## 📋 Yapılması Gerekenler

### 1. İçerik Optimizasyonu
- [ ] Her sayfada H1 başlık kullanımı optimize edilecek
- [ ] Alt text'ler (logo için) eklenecek
- [ ] İç linkler arttırılacak
- [ ] 300+ kelime içerik her section'a eklenebilir

### 2. Performans
- [ ] Görseller optimize edilmeli (WebP formatı)
- [ ] Lazy loading eklenmeli
- [ ] CSS/JS minification
- [ ] Gzip compression aktif edilmeli
- [ ] CDN kullanımı düşünülmeli

### 3. Teknik SEO
- [ ] SSL sertifikası (HTTPS)
- [ ] Google Search Console'a site eklenecek
- [ ] Google Analytics kurulumu
- [ ] Bing Webmaster Tools
- [ ] Page speed optimization
- [ ] Mobile-first indexing testi

### 4. Schema.org Güncellemeleri
- [ ] Gerçek iletişim bilgileri girilmeli
- [ ] Gerçek adres bilgileri
- [ ] Review ve rating bilgileri
- [ ] Sosyal medya linkleri

### 5. Local SEO
- [ ] Google My Business kaydı
- [ ] Yerel dizinlere kayıt
- [ ] NAP (Name, Address, Phone) tutarlılığı
- [ ] Müşteri yorumları toplama

### 6. Content Marketing
- [ ] Blog bölümü eklenmeli
- [ ] FAQ sayfası
- [ ] Referanslar/Projeler sayfası
- [ ] Video içerikler
